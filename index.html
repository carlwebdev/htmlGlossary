Code-related

Here are the HTML tags related to presenting code:

  • <code>: Defines a piece of computer code.
  • <pre>: Defines preformatted text.
  • <samp>: Defines sample output from a computer program.
  • <kbd>: Defines keyboard input.
  • <var>: Defines a variable.
  • <tt>: Defines teletype text (deprecated in HTML5).

Examples:

Example of <code>:

console.log('Hello, world!');

Example of <pre>:

    function helloWorld() {
        console.log('Hello, world!');
    }

Example of <samp>:

Output: Hello, world!

Example of <kbd>:

Ctrl + C

Example of <var>:

x = y + 2;

Example of <tt> (deprecated):

Teletype text

Media-related

Here are some HTML tags related to media:

  • <img>: Embeds an image.
  • <picture>: Contains zero or more <source> elements and one <img> element to provide multiple image formats.
  • <video>: Embeds a video.
  • <audio>: Embeds an audio file.
  • <iframe>: Embeds another HTML page.
  • <canvas>: Used to draw graphics via scripting (usually JavaScript).
